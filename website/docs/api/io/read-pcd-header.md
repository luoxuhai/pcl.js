# readPCDHeader

```ts
readPCDHeader(filename)
```

Read the header of any PCD file.

**Parameters:**

| Name     | Type     | Default | Description                   |
| -------- | -------- | ------- | ----------------------------- |
| filename | `string` |         | The name of the file to load. |

**Returns:**

| Name   | Type                   |
| ------ | ---------------------- |
| header | `{ fields: string[] }` |
