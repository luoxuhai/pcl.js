# readPCDHeader

```ts
readPCDHeader(filename)
```

读取任何 PCD 文件的 header

**参数:**

| 名称     | 类型     | 默认值 | 描述                 |
| -------- | -------- | ------ | -------------------- |
| filename | `string` |        | 要加载的文件的名称。 |

**返回:**

| 名称   | 类型                   |
| ------ | ---------------------- |
| header | `{ fields: string[] }` |
